# Laboratoires de virtualisation

Ce dépôt regroupe les laboratoires réalisés avec VirtualBox dans le cadre de mon apprentissage des systèmes, des réseaux et de la cybersécurité.

## Objectif

Mettre en place un environnement de test virtualisé afin de découvrir différents systèmes d'exploitation, comprendre les principes de la virtualisation et préparer les futurs laboratoires réseau et cybersécurité.

## Environnement utilisé

- VirtualBox
- Windows 11
- Ubuntu Linux
- Debian Linux
- Manjaro Linux
- Kali Linux

## Travaux réalisés

### Création de machines virtuelles

- Installation de plusieurs systèmes d'exploitation
- Configuration des ressources matérielles (RAM, processeur, stockage)
- Gestion des snapshots et de l'environnement de test

### Découverte de Linux

- Navigation dans les différents environnements Linux
- Prise en main des interfaces graphiques
- Familiarisation avec l'administration système

### Préparation des laboratoires réseau

- Mise en place d'environnements dédiés aux futurs travaux Cisco Packet Tracer
- Tests de compatibilité entre systèmes
- Préparation à l'apprentissage d'Active Directory et des services réseau

## Compétences développées

- Virtualisation
- Gestion de machines virtuelles
- Installation de systèmes d'exploitation
- Administration système débutante
- Découverte des distributions Linux
- Environnement de laboratoire

## Résultat

Mise en place d'un laboratoire VirtualBox fonctionnel permettant d'expérimenter plusieurs systèmes d'exploitation dans un environnement isolé et sécurisé afin de développer progressivement mes compétences en systèmes, réseaux et cybersécurité.

## Évolution du projet

Prochaines étapes :

- Administration Linux
- Active Directory
- DNS
- DHCP
- Routage et switching
- Sécurisation des systèmes
- Cybersécurité offensive et défensive
